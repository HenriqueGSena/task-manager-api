
# Gerenciador de Tarefas (Task Manager API)

Create a simple RESTful API to manage tasks, with CRUD functionalities and development best practices.

## Stack utilizada

**Back-end:** .Net Core 6, C#

**Banco de Dados:** Sqlite
## Rodando localmente

Clone the project

```bash
  git clone https://github.com/HenriqueGSena/task-manager-api.git
```

Enter the project directory

```bash
  cd task-manager-api
```

Running project

```bash
  dotnet run
```
